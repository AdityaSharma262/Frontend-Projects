# 🔐 Password Generator

A simple web app that generates strong, random passwords based on user-selected criteria like length and character types.

## 💡 Features

- Choose password length (e.g., 8 to 20 characters)
- Include uppercase, lowercase, numbers, and symbols
- Copy password to clipboard with one click

## 🛠 Tech Used

- HTML  
- CSS  
- JavaScript  

## 🚀 How to Run

Open `index.html` in any modern browser and start generating secure passwords!
