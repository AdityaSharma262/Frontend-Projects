# ❌⭕ Tic Tac Toe

A simple two-player Tic Tac Toe game with an interactive UI.

## 💡 Features

- Two-player gameplay  
- Highlights winning combination  
- Reset option to start a new game  

## 🛠 Tech Used

- HTML  
- CSS  
- JavaScript  

## 🚀 How to Run

Open `index.html` in a browser and enjoy the game.
