# 💻 Mini Projects Collection

Welcome to my personal collection of mini-projects built using HTML, CSS, and JavaScript. These projects were created to improve my frontend development skills and explore practical use-cases.

## 📂 List of Projects

### 🔢 [Age Calculator](./age-calculator)
A simple web app that calculates a person's age based on the input birth date.

### 🛒 [Amazon Clone (Frontend)](./amazon-clone)
A frontend-only replica of the Amazon homepage using HTML and CSS for layout and design.

### ➗ [Calculator](./calculator)
A responsive calculator that performs basic arithmetic operations using JavaScript.

### 🔐 [Password Generator](./password-generator)
Generates strong random passwords based on user-defined length and character preferences.

### ✊✋✌ [Rock Paper Scissors](./rock-paper-scissors)
A fun interactive Rock-Paper-Scissors game between the user and the computer.

### ❌⭕ [Tic Tac Toe](./tic-tac-toe)
A 2-player classic Tic Tac Toe game with a simple and intuitive UI.

### 📲 [Text to QR Code Generator](./txt-to-qr-code)
Converts plain text into a scannable QR code using a free JavaScript-based library.

### ✅ [To-Do List](./to-do-list)
A modern To-Do List application built using React and Next.js featuring dynamic UI and state management.
